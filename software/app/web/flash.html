MediStride — Flasher un module

Installer le firmware sans aucun logiciel

Cette page permet de programmer un module XIAO ESP32-S3 directement depuis votre navigateur, sans installer Arduino ni PlatformIO. Compatible Chrome et Edge sur ordinateur (pas iOS, pas Firefox).

1 Branchez le module XIAO ESP32-S3 à votre ordinateur avec un câble USB-C. Maintenez le bouton BOOT enfoncé, appuyez brièvement sur RESET, puis relâchez BOOT. Le module est maintenant en mode programmation.
2 Choisissez le firmware à installer selon le pied :

Pied gauche

Nom BLE : MediStride-L

Navigateur non compatible. Utilisez Chrome ou Edge. Vous devez accéder à cette page via HTTPS.

Pied droit

Nom BLE : MediStride-R

Navigateur non compatible. Utilisez Chrome ou Edge. Vous devez accéder à cette page via HTTPS.
3 Sélectionnez le port du module dans la fenêtre qui s'ouvre. Sur Linux, il s'agit généralement de /dev/ttyACM0 ; sur Windows de COM3 ou similaire ; sur macOS de /dev/cu.usbmodem*.
4 Attendez la fin du flashage (env. 30 secondes). Le module redémarre automatiquement et émet en BLE sous son nouveau nom.

Vous ne trouvez pas votre module dans la liste des ports ? Vérifiez le câble USB (certains ne transmettent que l'alimentation), réinstallez les pilotes CP210x (sur Windows) ou consultez le guide de dépannage.

Note : Les binaires utilisés ici sont publiés sur la page Releases du dépôt GitHub. Vous pouvez aussi flasher manuellement avec PlatformIO — voir le guide flashage.md.
